# streaming_stock_quotes

Приложение для отправки котировок с клиента серверу по udp. 
Сервер генерирует рандомные значения котировок и рассылает подключенным клиентам.
Клиент при запуске подключается к серверу и начинает получать котировки, при этом поддерживая соединение отправкой PING запросов.

#### Команда для запуска сервера
``` rust
cargo run --bin server crates/server/tickers.txt 
```
! Команду подготовлена с учетом выполнения из папки `streaming_stock_quotes`

В параметре передается путь до файла со списком котировок.

#### Команда для запуска клиента

``` rust
cargo run --bin client -- --udp-port 1112 --server-ip 127.0.0.1 --server-port 1111 --tickers-file crates/client/tickers1.txt 
```
! Команду подготовлена с учетом выполнения из папки `streaming_stock_quotes`

Параметры:
- udp-port - порт на котором клиент начнет ожидать upd соединения. 
- server-ip - ip адрес сервера. 
- server-port - порт сервера. Сервер всегда стартует на 1111 порту. Это порт для tcp подключения по которому клиент и сервер договорятся об установлении udp соединения.
- tickers-file - список котировок которые будут запрашиваться с сервера
